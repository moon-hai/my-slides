Text Editor - IDE

Tex Editor

  • Text Editor được biết đến là một công cụ làm việc với file/folder. Đặc điểm của nó là đơn giản, gọn nhẹ và không có màu sắc bắt mắt.

  • Một số Text Editor: Notepad, Notepad++, Sublime text, VScode,…​

IDE (Integrated Development Environment)

  • Là một môi trường thích hợp để viết code phát triển các ứng dụng.

  • Được tích hợp sẵn một số công cụ như Compiler, Interpreter, Debugger, định dạng, highlight, tổ chức thư mục, tìm kiếm,…​

  • Một số IDE phổ biến: Microsoft Visual Code, NetBeans, Eclipse, Xcode, Android Studio,…​

Giới thiệu Vim(Vi IMproved)

Vim

  • Là một bản sao và nâng cấp dựa trên trình soạn thảo vi (được viết bởi Bill Joy) cung cấp cho hệ điều hành Unix. Vim được viết bởi Bram Moolenaar và được phát hành lần đầu năm 1991 (WIKI)

  • Giao diện của Vim không dựa trên menu hay icon mà dựa trên các lệnh từ text user interface

Vim

  • Hiện nay đã có phiên bản NeoVim, là phần mở rộng của Vim, đang được sử dụng khá phổ biến.

  • Rất dễ nâng cấp từ Vim lên NeoVim vì cả hai sử dụng cú pháp và cấu hình giống nhau.

Tại sao di cư sang Vim

Why

  • Giảm thiểu bộ nhớ tiêu tốn cho dự án

  • Thời gian làm việc trên ternimal khá nhiều ⇒ dùng luôn cho tiện

  • Có thể build Vim như một IDE, custom những thứ cần thiết

  • Hạn chế sử dụng chuột để tăng tốc độ thao tác

  • Khi chuyển sang hdh khác (Linux) hoặc làm việc với server sẽ dễ dàng hơn

Hạn chế

  • Config rất tốn thời gian

  • Giao diện rất chán

  • Có rất nhiều phím cần phải nhớ để sử dụng

Một số bước set up cho Vim

Giới thiệu về các mode trong Vim

  • Normal Mode: Mặc định khi mở Vim ⇒ chỉnh sửa, sao chép, dán, xoá, di chuyển,…​

  • Insert Mode: Là chế độ để thay đổi text.

  • Visual Mode: Được sử dụng để lựa chọn text

  • Termial Mode: Mở terminal trong Vim.

Plugin Management

  • Dùng để quản lý Plugin cài đặt trong Vim

  • Một số Plugin management: Vunble(stop maintaince)), Patheon, Vim-Plug
